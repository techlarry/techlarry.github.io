数据结构介绍

概念

数据是一个抽象的概念，将其进行分类后得到程序设计语言中的基本类型。如：int，float，char等。数据元素之间不是独立的，存在特定的关系，这些关系便是结构。数据结构指数据对象中数据元素之间的关系。

程序= 数据结构+算法
总结：算法是为了解决实际问题而设计的，数据结构是算法需要处理的问题载体。

抽象数据类型(Abstract Data Type)

抽象数据类型(ADT)的含义是指一个数学模型以及定义在此数学模型上的一组操作。即把数据类型和数据类型上的运算捆在一起，进行封装。引入抽象数据类型的目的是把数据类型的表示和数据类型上运算的实现与这些数据类型和运算在程序中的引用隔开，使它们相互独立。

内存、类型本质、连续存储

内存是连续的存储空间，内存的基本存储单元是一个字节，1个字节(Byte)是8位(Bit)

顺序表的内存形式：

  • 逻辑地址：0,1,2,3,...,n01
  • 元素存储: e0,e1,e2,e3,....,en-1
  • 物理地址:l0,l0+c,l0+2c,l0+3c,l0+(n-1)c

调试

pdb

pdb是基于命令行的调试工具，是Python的调试器，非常类似GNU的GDB（调试c/c++）

python -m pdb some.py # -m是模块的意思, pdb模块

命令

  • 输入命令l (list) 查看代码
  • 输入命令n (next) 可以单步执行代码
  • 输入命令c (continue) 继续执行代码
  • 输入命令b (break) 添加断点
  • 输入命令clear 清除断点
  • 输入命令p (print) 打印一个变量的值
  • 输入命令s (step) 进入到一个函数
  • 输入命令a (args) 打印所有的形参数据
  • 输入命令r (return) 快速执行到函数的最后一行
  • 输入命令q quit 退出调试

闭包

在函数内部再定义一个函数，并且这个函数用到了外边函数的变量，那么将这个函数以及用到的一些变量统称为闭包

def test(number):
  def test_in(number_in):
    print("in test_in, number_in is %d"%number_in)
    return number+number_in
  return test_in

ret = test(100)  # ret 指向函数体
ret(1)  # 调用函数

闭包的实际例子：

def line_conf(a,b):
  def line(x):
    return a*x + b
  return line

line1 = line_conf(1,1)
line2 = line_conf(4,5)

print(line1(5))
print(line2(5))
  1. 闭包又花了变量，原来需要类对象完成的工作，闭包也可以完成
  2. 由于必报引用了外部函数的局部变量，则外部函数的局部变量没有及时释放，消耗内存

装饰器

装饰器是程序开发中经常会用到的一个功能，用好了装饰器，开发效率如虎添翼。

  • 只要python解释器执行到了装饰器代码部分，那么就会自动装饰，而不是等到调用的时候才装饰
  • 有多个装饰器时，先执行后面的装饰
def w1(func):
  print('---正在装饰1-----')
  def inner():
    print('-----装在验证权限1------')
    func()
  return inner
  
def w2(func):
  print('---正在装饰2-----')
  def inner():
    print('-----装在验证权限2------')
    func()
  return inner

@w1
@w2
def f1():
  print("---f1---")

f1()

>>>
---正在装饰2-----
---正在装饰1-----
-----装在验证权限1------
-----装在验证权限2------
---f1---

通用装饰器

无参数、有参数、不定长参数、有返回值、无返回值 示例

def func(functionName):

  def func_in(*args, **kwargs):
    ret = functionName(*args, **kwargs)
    return ret
    
  return func_in

带有参数的装饰器

同一个装饰器，在装饰不同函数时，起到不同的效果

def func_arg(arg):
    def func(functionName):
        def func_in():
            print("---记录日志-arg=%s--"%arg)
            if arg=="heihei":
                functionName()
                functionName()
            else:
                functionName()
        return func_in
    return func

#1. 先执行func_arg("heihei")函数,这个函数return 的结果是func这个函数的引用
#2. @func
#3. 使用@func对test进行装饰
@func_arg("heihei")
def test():
    print("--test--")

#带有参数的装饰器,能够起到在运行时,有不同的功能
@func_arg("haha")
def test2():
    print("--test2--")

test()
test2()

python是动态语言

C语言是静态语言，运行之前需要先编译，不允许在运行过程中修改代码。

动态编程语言，在运行时可以改变其结果的语言：例如新的函数、对象、甚至代码可以被引进，已有的函数可以被删除或是其他结构上的变化。

运行的过程中给实例/类添加属性(实例属性/类属性)

class Person(object):
    def __init__(self, name=None, age=None):
             self.name = name
             self.age = age
P = Person('X man','24')
P.sex = 'male'  # 给对象添加属性
Person.sex = 'female' #给类添加属性

运行的过程中给实例/类添加方法

给实例添加方法时，一般需使用MethodType函数。

class Person(object):
    def __init__(self, name=None, age=None):
             self.name = name
             self.age = age
    def eat(self):
        print('---%s is eating'%self.name)
        
def run(self):
    print('---%s is running'%self.name)
    
p1= Person('P1', 10)
p1.eat()
p1.run = run
#虽然p1对象中 run属性已经指向了第10行的函数，但是这句代码还不正确
#因为run属性的函数是后来添加的，即p1.run()的时候，并没有把p1当作第1个参数

import types
p1.run = types.MethodType(run,p1)
p1.run()

但是静态方法(static method)和类方法(class method)可以直接添加

@staticmethod
def test():
    print("---static method-----")
p1.test = test
p1.test()

@classmethod
def printNum(cls):
    print('----class method---')
    
Person.printNum = printNum
Person.printNum()

限制属性 __slots__

定义一个特殊的__slot__变量，限制class实例能添加的属性

class Person(object):
    __slots__=("name","age")

p = Person()
p.name = "老王"
p.age = 10
p.score = 100
