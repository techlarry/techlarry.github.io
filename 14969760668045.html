闭包，装饰器

2017/4/5 posted in  Python特性

函数引用

>>> def test():
...     print('lala')
...
>>> test()
lala
>>> test
<function test at 0x109a51950> 

>>> b = test  # b指向函数体test

>>> b
<function test at 0x109a51950>

>>> b()  # 通过引用调用函数
lala

闭包

在函数内部再定义一个函数，并且这个函数用到了外边函数的变量，那么将这个函数以及用到的一些变量统称为闭包

def test(number):
  def test_in(number_in):
    print("in test_in, number_in is %d"%number_in)
    return number+number_in
  return test_in

ret = test(100)  # ret 指向函数体
ret(1)  # 调用函数

闭包的实际例子：

def line_conf(a,b):
  def line(x):
    return a*x + b
  return line

line1 = line_conf(1,1)
line2 = line_conf(4,5)

print(line1(5))
print(line2(5))
  1. 闭包又花了变量，原来需要类对象完成的工作，闭包也可以完成
  2. 由于必报引用了外部函数的局部变量，则外部函数的局部变量没有及时释放，消耗内存

装饰器

装饰器是程序开发中经常会用到的一个功能，用好了装饰器，开发效率如虎添翼。
* 只要python解释器执行到了装饰器代码部分，那么就会自动装饰，而不是等到调用的时候才装饰
* 有多个装饰器时，先执行后面的装饰

def w1(func):
  print('---正在装饰1-----')
  def inner():
    print('-----装在验证权限1------')
    func()
  return inner
  
def w2(func):
  print('---正在装饰2-----')
  def inner():
    print('-----装在验证权限2------')
    func()
  return inner

@w1
@w2
def f1():
  print("---f1---")

f1()

>>>
---正在装饰2-----
---正在装饰1-----
-----装在验证权限1------
-----装在验证权限2------
---f1---

通用装饰器

无参数、有参数、不定长参数、有返回值、无返回值 示例

def func(functionName):

  def func_in(*args, **kwargs):
    ret = functionName(*args, **kwargs)
    return ret
    
  return func_in

带有参数的装饰器

同一个装饰器，在装饰不同函数时，起到不同的效果

def func_arg(arg):
    def func(functionName):
        def func_in():
            print("---记录日志-arg=%s--"%arg)
            if arg=="heihei":
                functionName()
                functionName()
            else:
                functionName()
        return func_in
    return func

#1. 先执行func_arg("heihei")函数,这个函数return 的结果是func这个函数的引用
#2. @func
#3. 使用@func对test进行装饰
@func_arg("heihei")
def test():
    print("--test--")

#带有参数的装饰器,能够起到在运行时,有不同的功能
@func_arg("haha")
def test2():
    print("--test2--")

test()
test2()
