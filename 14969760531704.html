python是动态语言

2017/6/6 posted in  Python特性

C语言是静态语言，运行之前需要先编译，不允许在运行过程中修改代码。

动态编程语言，在运行时可以改变其结果的语言：例如新的函数、对象、甚至代码可以被引进，已有的函数可以被删除或是其他结构上的变化。

运行的过程中给实例/类添加属性(实例属性/类属性)

class Person(object):
    def __init__(self, name=None, age=None):
             self.name = name
             self.age = age
P = Person('X man','24')
P.sex = 'male'  # 给对象添加属性
Person.sex = 'female' #给类添加属性

运行的过程中给实例/类添加方法

给实例添加方法时，一般需使用MethodType函数。

class Person(object):
    def __init__(self, name=None, age=None):
             self.name = name
             self.age = age
    def eat(self):
        print('---%s is eating'%self.name)
        
def run(self):
    print('---%s is running'%self.name)
    
p1= Person('P1', 10)
p1.eat()
p1.run = run
#虽然p1对象中 run属性已经指向了第10行的函数，但是这句代码还不正确
#因为run属性的函数是后来添加的，即p1.run()的时候，并没有把p1当作第1个参数

import types
p1.run = types.MethodType(run,p1)
p1.run()

但是静态方法(static method)和类方法(class method)可以直接添加

@staticmethod
def test():
    print("---static method-----")
p1.test = test
p1.test()

@classmethod
def printNum(cls):
    print('----class method---')
    
Person.printNum = printNum
Person.printNum()

限制属性 __slots__

定义一个特殊的__slot__变量，限制class实例能添加的属性

class Person(object):
    __slots__=("name","age")

p = Person()
p.name = "老王"
p.age = 10
p.score = 100
>>>> AttributeError: 'Person' object has no attribute 'score'
