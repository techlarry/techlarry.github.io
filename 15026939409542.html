逻辑删除

2017/8/14 posted in  数据库

逻辑删除：当数据非常重要时，在实际执行删除操作时，只是将数据标记为删除。在数据查询操作时，加上是否删除语句。

mysql> alter table birds add column isDelete bit default b'0';
mysql> update birds
    -> set isDelete = b'1'
    -> where bird_id = 101;
mysql> select count(*)
    -> from birds
    -> where `isDelete` = 0;
+----------+
| count(*) |
+----------+
| 28890    |
+----------+
