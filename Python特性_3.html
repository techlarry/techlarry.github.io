闭包

在函数内部再定义一个函数，并且这个函数用到了外边函数的变量，那么将这个函数以及用到的一些变量统称为闭包

def test(number):
  def test_in(number_in):
    print("in test_in, number_in is %d"%number_in)
    return number+number_in
  return test_in

ret = test(100)  # ret 指向函数体
ret(1)  # 调用函数

闭包的实际例子：

def line_conf(a,b):
  def line(x):
    return a*x + b
  return line

line1 = line_conf(1,1)
line2 = line_conf(4,5)

print(line1(5))
print(line2(5))
  1. 闭包又花了变量，原来需要类对象完成的工作，闭包也可以完成
  2. 由于必报引用了外部函数的局部变量，则外部函数的局部变量没有及时释放，消耗内存

装饰器

装饰器是程序开发中经常会用到的一个功能，用好了装饰器，开发效率如虎添翼。

  • 只要python解释器执行到了装饰器代码部分，那么就会自动装饰，而不是等到调用的时候才装饰
  • 有多个装饰器时，先执行后面的装饰
def w1(func):
  print('---正在装饰1-----')
  def inner():
    print('-----装在验证权限1------')
    func()
  return inner
  
def w2(func):
  print('---正在装饰2-----')
  def inner():
    print('-----装在验证权限2------')
    func()
  return inner

@w1
@w2
def f1():
  print("---f1---")

f1()

>>>
---正在装饰2-----
---正在装饰1-----
-----装在验证权限1------
-----装在验证权限2------
---f1---

通用装饰器

无参数、有参数、不定长参数、有返回值、无返回值 示例

def func(functionName):

  def func_in(*args, **kwargs):
    ret = functionName(*args, **kwargs)
    return ret
    
  return func_in

带有参数的装饰器

同一个装饰器，在装饰不同函数时，起到不同的效果

def func_arg(arg):
    def func(functionName):
        def func_in():
            print("---记录日志-arg=%s--"%arg)
            if arg=="heihei":
                functionName()
                functionName()
            else:
                functionName()
        return func_in
    return func

#1. 先执行func_arg("heihei")函数,这个函数return 的结果是func这个函数的引用
#2. @func
#3. 使用@func对test进行装饰
@func_arg("heihei")
def test():
    print("--test--")

#带有参数的装饰器,能够起到在运行时,有不同的功能
@func_arg("haha")
def test2():
    print("--test2--")

test()
test2()

python是动态语言

C语言是静态语言，运行之前需要先编译，不允许在运行过程中修改代码。

动态编程语言，在运行时可以改变其结果的语言：例如新的函数、对象、甚至代码可以被引进，已有的函数可以被删除或是其他结构上的变化。

运行的过程中给实例/类添加属性(实例属性/类属性)

class Person(object):
    def __init__(self, name=None, age=None):
             self.name = name
             self.age = age
P = Person('X man','24')
P.sex = 'male'  # 给对象添加属性
Person.sex = 'female' #给类添加属性

运行的过程中给实例/类添加方法

给实例添加方法时，一般需使用MethodType函数。

class Person(object):
    def __init__(self, name=None, age=None):
             self.name = name
             self.age = age
    def eat(self):
        print('---%s is eating'%self.name)
        
def run(self):
    print('---%s is running'%self.name)
    
p1= Person('P1', 10)
p1.eat()
p1.run = run
#虽然p1对象中 run属性已经指向了第10行的函数，但是这句代码还不正确
#因为run属性的函数是后来添加的，即p1.run()的时候，并没有把p1当作第1个参数

import types
p1.run = types.MethodType(run,p1)
p1.run()

但是静态方法(static method)和类方法(class method)可以直接添加

@staticmethod
def test():
    print("---static method-----")
p1.test = test
p1.test()

@classmethod
def printNum(cls):
    print('----class method---')
    
Person.printNum = printNum
Person.printNum()

Python 类属性与实例属性

Python属于动态语言，类属性很好体现了python动态语言这一特性。
对于属性，通常可以通过类属性和实例属性的形式调用。
看一个例子：

class AAA():
    aaa = 10
    
obj1 = AAA()
obj2 = AAA()
print(obj1.aaa, obj2.aaa, AAA.aaa) # result: 10 10 10

obj1.aaa += 2
print(obj1.aaa, obj2.aaa, AAA.aaa) # result: 12 10 10

AAA.aaa += 4
print(obj1.aaa,obj2.aaa,AAA.aaa)  # result: 12 14 14

上例中，AAA.aaa就属于类属性形式，obj1.aaa属于实例属性的形式
